git config --global user.name blabla

- встановлює ім'я користувача, що відображатиметься у інформацї про виконані зміни

git config --global user.email blabal@bla - встановлює email користувача, що відображатиметься у інформацї про виконані зміни
git config --global core.editor "''" - задає редактор для формування повдомлень
git config --list - виводить перелік налаштувань поточного репозиторія git init - створення репозиторію поточної директорії (директорія вказується командою cd "path")
git status - виводить перелік індексованих файлів, що були модифіковані
git add - додає файли у перелік який буде комітитись
git commit - комітить зміни, що були включені командою git add
git log - виводить перелік комітів
git checkout - переключається на вказаний (по унікальному ідентифікатору) коміт та завантажує тодішній стан проекту у робочу директорію проекту
git branch
git push
git pull
git merge
git clone
